Interstellar Web Proxy

This project is designed to run with Node.js and Express. Follow the steps below to run the project locally:

  1. Clone the repository:
    2. git clone https://github.com/UseInterstellar/Interstellar.git
  2. Navigate to the project directory:
    3. cd Interstellar
  3. Install dependencies using pnpm:
    4. pnpm install
  4. Start the server:
    5. npm run start

After running these commands, the server will be up and running on http://localhost:3000.
